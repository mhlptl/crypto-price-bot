# Discord Bot: Crypto Price

This Discord Bot returns the current price of the cryptocurrency the user has asked about

## Commands

The commands the bot currently understands are as follows:

* **crypto help** -> this displays the commands that the bot is able to comprehend
* **$TICKER** -> this returns the current price of the cryptocurrency the user has asked about (i.e $BTC returns the price of Bitcoin)

Built using the [discord.js](https://discord.js.org/#/) library
